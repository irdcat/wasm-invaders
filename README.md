# wasm-invaders

## Local development

Prequisites:
- emscripten
- node
- make
- cmake

### Installing http-server

`npm install -g http-server`

## Building & Running emulator

Use `build` script

### Serving generated page

`http-server ./`

### Run

Open localhost:8080/wasm-invaders.html in the browser

## Building & Running tests

Use `build-tests` script
